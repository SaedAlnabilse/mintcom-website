# PayMint AI Knowledge Base

This document serves as the comprehensive knowledge source for the PayMint AI Chatbot. It covers the product profile, features, pricing, operations, and technical details.

## 1. Product Overview

**What is PayMint?**
PayMint is a comprehensive, cloud-based Point of Sale (POS) system designed specifically for **restaurants, cafes, and retail establishments**. It aims to streamline daily operations by eliminating the complexity of traditional POS systems, offering an intuitive interface for sales, inventory, staff management, and analytics.

**Target Audience:**
- Restaurants
- Cafes
- Retail Stores
- Franchises and multi-location enterprises

**Core Value Proposition:**
- **All-in-One Solution:** Manage sales, inventory, and staff from one dashboard.
- **Lightning Fast:** Optimized for speed and ease of use.
- **Cloud-Based:** Real-time synchronization across all devices.
- **Secure:** Enterprise-grade encryption.
- **Hardware Agnostic:** Works with most Android tablets (8"+) and iPads.

## 2. Key Features

### 2.1 Dashboard & Analytics
The central command center providing real-time business insights.
- **Key Metrics:** Total Sales, Order Counts, Cash vs. Card Sales, Petty Cash (PAY-IN/PAY-OUT), Labor Hours.
- **Visuals:** Hourly sales trend charts.
- **Actions:** Quick access to Cash In/Out and End Shift functions.
- **Reports:**
    - **Sales Summaries:** Daily, weekly, monthly.
    - **Item Performance:** Top-selling products.
    - **Peak Hours:** Traffic analysis for staffing.
    - **Export:** PDF, Email, or Print directly from the app.

### 2.2 Point of Sale (POS) Operations
- **Order Processing:**
    - Visual category and product selection.
    - **Modifiers/Add-ons:** Easy customization (e.g., "No ice", "Extra shot").
    - **Notes:** Special instructions for kitchen/staff.
    - **Holding Orders:** Pause and resume transactions (e.g., for table service).
- **Payments:**
    - **Multiple Methods:** Cash, Card, Third-party apps (Talabat, Careem).
    - **Split Payments:** By amount or by item.
    - **Discounts:** Preset percentages, fixed amounts, or custom item-level discounts.
- **Refunds:** Full or partial refunds with mandatory reason logging and optional admin authorization.

### 2.3 Inventory Management
- **Product Management:** Add/Edit items with photos, prices, costs, and SKUs.
- **Stock Tracking:** Real-time deduction with low-stock alerts.
- **Categories:** Organize products for faster checkout.
- **Attributes:** Manage add-on groups (e.g., Sizes, Toppings).

### 2.4 Staff Management
- **Roles:**
    - **Admin:** Full access to settings and reports.
    - **User:** Restricted access based on permissions.
- **Security:** 4-digit PIN login for fast user switching.
- **Shift Management:**
    - Clock-in/Clock-out tracking.
    - **Z-Report:** End-of-shift reconciliation with cash counting and discrepancy logging.
- **Activity Log:** Audit trail of all sensitive actions (refunds, price changes, etc.).

### 2.5 Customer & Loyalty
- **Profiles:** CRM to track customer history and contact info.
- **Loyalty Program:** Built-in points system (earn and redeem).

## 4. Hardware Compatibility

PayMint is designed to run on standard consumer hardware.

**Tablets (POS Terminal):**
- **Requirements:** Any Android tablet (8"+ screen) or iPad.
- **Recommended Models:**
    - Samsung Galaxy Tab A8 (~$180)
    - iPad 10th Gen (~$350)
    - Lenovo Tab M10 Plus (~$150)

**Receipt Printers:**
- **Requirements:** Most ESC/POS thermal printers (80mm or 58mm).
- **Recommended Models:**
    - Munbyn Thermal Printer (USB+Bluetooth)
    - Epson TM-T20III (USB/Ethernet)
    - Star Micronics TSP143 (Bluetooth)

## 5. Technical & Integration

### 5.1 Website Widget Integration
PayMint provides a widget that can be embedded into external websites.
**Installation:**
Add the following script tag before the closing `</body>` tag of the target website:
```html
<script src="https://sa3d100-PayMint-test.hf.space/widget.js"></script>
```

### 5.2 System Requirements
- **Connectivity:** Stable Wi-Fi connection is required for real-time sync.
- **Browsers:** Optimized for modern web browsers (Chrome, Safari, Edge).

### 5.3 Troubleshooting Common Issues
- **Connection Failed:** Check Wi-Fi.
- **Printer Not Found:** Ensure printer is powered on and paired via Bluetooth or on the same network.
- **Shift Conflicts:** Only one user can have an active shift on a specific register at a time.
