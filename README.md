# Mintcom Website

Vite/React website for Mintcom public pages, web content, downloadable assets, and deployment-facing frontend workflows.

## Setup

```powershell
npm install
```

## Development

```powershell
npm run dev
```

## Build

```powershell
npm run build
```

## Tests

```powershell
npm test
npm run test:e2e
```

## Deploy

```powershell
npm run deploy
```

## Notes

This folder replaces the old `PaymintWebsite` path. The npm package is `mintcom-website`.
